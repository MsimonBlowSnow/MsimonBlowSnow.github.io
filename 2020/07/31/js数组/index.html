js数组常用的方法

1.push，pop，shift，unshift 均会改变原数组

​ push: 从尾部加入一个元素,相当于 arr[arr.length] = item;

​ pop: 从尾部删除一个数组元素，并返回该元素，

​ shift；从头部删除一个数组元素，并返回该元素

​ unshift: 从头部添加一个元素

2.reverse,sort 会改变数组的元素排序

reverse 将数组反序

​ sort 将数组排序

​ 默认: 是将每个元素转化为字符串，在进行排序，(升序排序)

​ 例如:

sort可以传入一个方法，方法返回正数，顺序不变，方法返回一个负数，顺序改变。

3.concat,slice(不会改变原数组)

使用concat(),将传入的参数追加到数组的后面，并返回追加完成的数组，注意原有的数组不会发生改变

使用slice截取一段数组，不会改变数组，第一个参数是截取数组的开始下标，第二个参数不给默认是截取到数组的尾部，给的话就是截取到的下边位置(不包括给位置)。

4.splice改变原来的素组

一、删除的用法

语法: array.splice(start,n);

​ start 指的是从哪个位置开始(索引，包括该下边的索引)

​ n指的是需要删除的个数

二、插入的用法

语法：array.splice(starti,0,值1，值2…);

starti: 在哪个位置插入，原来start位置的值向后顺移
0：表示删除0个元素，因为插入和替换都是由删除功能拓展的。
值1，值2：需要插入的值

三、替换方法

语法:array.splice(starti,n,值1，值2);
原理和插入的用法相同
实际是就是：在starti的位置删除n个元素，然后在这个位置插入值1，值2,就可以起到替换
